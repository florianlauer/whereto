### AUTH-05 Gap Detail

**Requirement:** "Auth proposée au moment du Save, jamais en gate sur la carte"

Phase 2 built `useAuthGatedAction` hook specifically for this. Phase 2 verification marked it SATISFIED based on hook existence + unit tests. However, integration check reveals the hook is **never imported in any production component**. `DestinationPanel.togglePoi` calls `addToWishlist` directly — anonymous users silently save to localStorage without being prompted to sign in.

**Fix:** Import `useAuthGatedAction` in `DestinationPanel`, wrap `togglePoi` body with `gateAction(...)`. Same for any other wishlist consumer components (ComparisonDrawer, TripSummaryPanel).
